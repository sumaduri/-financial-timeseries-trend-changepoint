# Financial Time-Series: Trend and Change-Point Analysis

This repository contains the R and Python scripts used in my master's thesis.  
The analysis includes:

- Linear Regression (R)
- Mann–Kendall Test (R)
- Pettitt Test (R)
- CUSUM Test (R)
- Linear Kernel Change-Point Detection (Python)

Data used: Monthly stock prices from 12 U.S. companies (2015–2025).
